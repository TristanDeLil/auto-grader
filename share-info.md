# Share changes

## Push changes

1. Make fork
2. Work in fork
3. Push changes:

```commands
git add .
git commit-m "message"
git push origin main
```

4. Create pull request (On github from your own account -  not the vives group)

## Pull changes

1. Merge request must be accepted first
2. In own repo:

```commands
git pull origin main
```
